# ForgeAI: Startup Elite Edition

## 💎 A Nova Era da Engenharia de Prompts
O **ForgeAI** (anteriormente PromptForge Pro) foi completamente reconstruído para atender aos padrões de startups de IA de elite do Vale do Silício. Abandonamos o padrão genérico para uma estética **Obsidian Black** com foco em performance e flexibilidade total.

## 🚀 Diferenciais de Elite
1.  **Infraestrutura Universal**: Não estamos presos a uma IA. Você pode conectar **qualquer modelo** via OpenRouter, OpenAI, Anthropic ou endpoints locais (Ollama) diretamente nas configurações.
2.  **Zero Mocks**: Todas as chamadas de IA são funcionais e processadas através de um cliente HTTP de alta performance escrito em **Rust (Tauri)**.
3.  **Estética Cyber-Minimalista**: Design focado em Dark Mode premium, utilizando Glassmorphism, ruído de fundo e micro-animações de alta fidelidade.
4.  **Neural Workspace**: Dashboard com estatísticas reais de consumo de tokens e latência de execução.
5.  **Prompt Canva**: Editor de prompt avançado com injeção modular e preview em tempo real.

## 🛠️ Stack Tecnológica
- **Backend**: Rust + Tauri (Segurança e Performance).
- **Frontend**: React 18 + Vite + Tailwind CSS (Custom Design System).
- **Engine**: Motor de prompt modular com suporte a injeção de variáveis e contexto de arquivos (PDF/DOCX/XLSX).

## 📦 Inicialização
```bash
# Clone e Instalação
pnpm install

# Iniciar Lab de Desenvolvimento
pnpm dev
```

## 🔒 Privacidade
Suas chaves de API são armazenadas localmente no seu dispositivo e as chamadas são feitas diretamente do seu computador para o provedor de IA, garantindo 100% de privacidade dos dados.
